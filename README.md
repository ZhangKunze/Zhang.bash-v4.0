# Zhang.bash-v4.0
English Edition：
This repository contains version 4.0 of the Zhang.bash script.
It offers an improved experience and upgraded functionality.
This version was developed locally using Git during the winter break when I couldn't log in to my GitHub account.
Versions 2.0, 3.0, and 3.2 have been modified over time, and I no longer have copies of them. They were unstable and had incomplete features.

Command:
```bash
chmod +x ~/Zhang.bash/Zhang.bash && .~/Zhang.bash/Zhang.bash
```

```bash
cd ~/Zhang.bash/ && chmod +x Zhang\(Not_good-Editon\).bash && ./Zhang\(Not_good-Editon\).bash
```

```bash
cd ~/Zhang.bash/ && chmod +x zkz && cp -a zkz $PATH/zkz
```
